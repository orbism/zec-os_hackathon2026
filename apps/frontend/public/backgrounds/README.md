# Desktop Backgrounds

Add the following background images to this directory:

- `bg1.jpg` - Alternative background image
- `bg2.jpg` - Alternative background image 
- 'bg3.jpg' - Default background image

## Recommended Specifications

- Format: JPEG
- Resolution: 1920x1080 minimum (larger is fine, will be scaled)
- File size: Keep under 500KB for faster loading
- Style: Should complement the retro OS aesthetic

## Usage

Users can select these backgrounds from:
1. Settings > Background > Builtin
2. Themes app
